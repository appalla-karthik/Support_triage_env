# Support Triage Docs

This workspace root now hosts a Docusaurus documentation site for the
`Support_triage_env` project.

The site documents:

- the project architecture and environment loop
- the reward and grading logic
- the training and submission workflow
- the best Round 2 hackathon theme fit
- a concrete pivot strategy to make the submission stronger and more unique

## Run locally

```bash
npm.cmd start
```

## Build

```bash
npm.cmd run build
```
